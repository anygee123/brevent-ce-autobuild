# 黑域社区版编译脚本

## 使用说明

下载后将`apksigner.jar`和`autogen.sh`放到黑域仓库目录下，在终端中执行

```
chmod +x autogen.sh
./autogen.sh
```

## 系统需求

需要安装 JDK、Gradle、Git 和 Android SDK。

建议在Linux下使用，但是你也可以在Windows下通过Msys或者其他终端来执行此脚本。

仅在msys2下测试过，不保证其他系统也可以使用。
